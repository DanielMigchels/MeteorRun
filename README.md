# MeteorRun

A Windows Forms space shooter game built with C# and .NET Framework 4.5.

*This project was created in 2017 during my studies as a learning project.*

## Description

MeteorRun is a classic arcade-style space shooter where you pilot a spaceship through space, dodging meteors, fighting enemies, and collecting power-ups. The game features multiple game states including a menu system, gameplay, highscores, instructions, and a shop.

## How to Play

1. Launch the application
2. Navigate through the menu using keyboard controls
3. Start a new game and control your spaceship
4. Avoid meteors and enemies while shooting them down
5. Collect power-ups to gain advantages
6. Survive as long as possible and achieve high scores
